# Organic Chicken Sales Management System

This application is a desktop-based sales management system developed using **Java Swing** and **MySQL**. It is designed to manage organic chicken sales, inventory restocking, and sales statistics in an efficient and structured manner.

The system helps streamline sales recording, stock monitoring, and provides statistical insights to support business decision-making.
